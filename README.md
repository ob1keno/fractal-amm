# Fractal AMM

Реализация фрактального автоматизированного маркет-мейкера с многослойной структурой.

## Особенности

- **Многослойная архитектура**: Вложенные пулы ликвидности
- **Адаптивное исполнение**: Ордера автоматически распределяются по слоям
- **Визуализация**: Готовые инструменты для анализа кривых ликвидности
